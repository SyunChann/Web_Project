# Review Collection

영화, 애니, 게임, 드라마처럼 본 콘텐츠를 기록하고 별점과 감상평을 남기는 개인 리뷰 컬렉션 웹입니다.

## 사용 기술

- Next.js
- React
- TypeScript
- Tailwind CSS

## 현재 구현된 기능

- 홈 화면
- 리뷰 목록 페이지
- 리뷰 상세 페이지
- 샘플 리뷰 데이터
- 반응형 기본 UI

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 아래 주소를 열면 됩니다.

```txt
http://localhost:3000
```

## 주요 폴더

```txt
src/app
페이지 라우팅과 화면 구성

src/data
샘플 리뷰 데이터
```
